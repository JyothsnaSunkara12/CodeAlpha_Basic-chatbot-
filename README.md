 Basic Rule-Based Chatbot

# Overview
This is a simple rule-based chatbot built using Python. It interacts with users by recognizing predefined inputs and responding accordingly. This project is designed to demonstrate fundamental programming concepts in a practical way.


# Features
- Responds to basic user inputs (hello, how are you, bye)
- Uses simple conditional logic (if-elif statements)
- Continuous conversation using loops
- Clean and beginnerfriendly code structure
